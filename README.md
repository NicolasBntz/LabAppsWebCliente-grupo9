# LabAppsWebCliente-grupo9

## Descripción
Este proyecto es una aplicación web cliente desarrollada como parte del curso de Laboratorio de Aplicaciones Web Cliente en ISTEA. El objetivo principal es implementar una aplicación funcional que utilice tecnologías modernas de desarrollo web.

## Estructura del Proyecto
El proyecto está organizado de la siguiente manera:

```
index.html
css/
    style.css
src/
    index.js
    components/
        cards.js
        cart.js
        contador.js
        modal.js
        toast.js
    services/
        api.js
    storage/
        storage.js
    utils/
        category-filters.js
```

### Archivos y Carpetas Principales
- **index.html**: Archivo principal de la aplicación.
- **css/style.css**: Contiene los estilos de la aplicación.
- **src/**: Carpeta que contiene el código fuente de la aplicación.
  - **components/**: Componentes reutilizables como tarjetas, carrito, contador, modales y notificaciones.
  - **services/**: Servicios para interactuar con APIs externas.
  - **storage/**: Módulos relacionados con el almacenamiento local.
  - **utils/**: Funciones utilitarias como filtros de categorías.

## Tecnologías Utilizadas
- **HTML5**
- **CSS3**
- **JavaScript (ES6+)**

## Instalación y Uso
1. Clona este repositorio:
   ```bash
   git clone <URL_DEL_REPOSITORIO>
   ```
2. Abre el archivo `index.html` en tu navegador para visualizar la aplicación.
